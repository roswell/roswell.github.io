GitHub Actions

GitHub Actions is an automation service by GitHub. Its notable advantages are seamless GitHub integration and good OS support -- Ubuntu, macOS and Windows.

It's still in limited public beta and the API is likely to change. We're working on its testing to provide better experience with GitHub Actions, but please remember its still quite experimental.

Getting Started

Add a workflow YAML file at .github/workflows/ci.yaml something like this:

name: CI

on: [push]

jobs:
  test:
    name: ${{ matrix.lisp }} on ${{ matrix.os }}
    runs-on: ${{ matrix.os }}
    strategy:
      matrix:
        lisp: [sbcl-bin]
        os: [ubuntu-latest, macOS-latest]

    steps:
      - uses: actions/checkout@v1
      - name: Install Roswell
        env:
          LISP: ${{ matrix.lisp }}
        run: |
          curl -L https://raw.githubusercontent.com/roswell/roswell/master/scripts/install-for-ci.sh | sh
      - name: Install Rove
        run: ros install rove
      - name: Run tests
        run: |
          PATH="~/.roswell/bin:$PATH"
          rove qlot.asd

Known issues

  • The latest install-for-ci.sh is required until 158bdeb6 will have been released.
  • [Windows] ROSWELL_INSTALL_DIR has to be set /usr on Windows because /usr/local/bin is not in PATH.
  • [Windows] Roswell scripts can't be executed with cmd.exe
  • Execute them through ros command like ros %userprofile%\.roswell\bin\rove
  • Use bash by setting jobs.<job-id>.steps.shell to bash.
  • [Windows] ASDF configuration is not set
  • Set the source registry by yourself like ros -S . -s rove -e ...
